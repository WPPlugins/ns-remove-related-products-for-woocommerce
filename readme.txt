Remove Related Products from your shop page

== Description ==

**Insstall plugin to remove related products from your shop page**

Easy to use, you only install the plugin.


*We test our plugins with many other. If you find incompatibility with one specific plugin, please inform us at <a href="mailto:info@nsthemes.com">info@nsthemes.com</a>.*

== Installation ==

1. Upload the entire 'ns-remove-related-product-for-woocommerce' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
